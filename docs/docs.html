Everything you need to know about using Linkivo effectively.

What is Linkivo?

Linkivo is a smart, offline-first link manager that runs in your browser as a Progressive Web App (PWA). Save links from any source, organise them into folders, and rediscover them through a weighted random system.

Installing as an app

Linkivo is a PWA — install it for the best experience:

  • Chrome/Edge (desktop): Click the install icon in the address bar, or go to Settings → Install Linkivo
  • Chrome (Android): Tap the three-dot menu → Add to Home Screen
  • Safari (iPhone/iPad): Tap Share → Add to Home Screen

Once installed, Linkivo works offline and loads instantly.

Importing links

Tap the + button (mobile) or Import Links (sidebar) to open the import modal.

Supported formats

  • .txt — one URL per line, or any text with URLs mixed in
  • .html / .htm — browser bookmark exports (NETSCAPE format)
  • .json — Linkivo exports, or any JSON with URL fields
  • .csv — spreadsheets with URL columns
  • .pdf — extracts URLs from PDF text content
  • .zip — extracts links from all supported files inside
  • .md — Markdown files with [title](url) syntax

Linkivo also detects URLs without https:// — e.g., example.com or www.site.io/page.

Folders & PIN lock

Create folders to organise links. Each folder can be:

  • Pinned — appears at the top of the list
  • Locked — requires a 6-digit PIN to view; once unlocked it stays open until the page reloads
  • Tagged — add custom tags for quick identification

To set a PIN lock, tap the ⋮ menu on a folder → Lock with PIN.

Random discover

The random page uses a weighted selection algorithm. Links with higher points (from likes, stars, and low open counts) are more likely to be picked. You can customise:

  • Folder filter: choose which folders to include
  • Domain filter: auto-detected from your links — filter to specific sites
  • Weight mode: by points, equal chance, liked links, or favourites only
  • Auto-advance: automatically pick a new link every 10–60 seconds

Link preview

When you open a link, Linkivo tries to load a preview using this fallback chain:

  1. iframe embed — loads the actual page inline (if the site allows it)
  2. og:image — fetches the OpenGraph preview image via a proxy
  3. First JPEG — uses the first jpg/jpeg image found in the page
  4. Favicon — shows the site's favicon as a last resort

Drag the resize handle at the top of the preview panel to adjust its height. The height is saved for next time.

Exporting your data

Go to Settings → Export Links. Choose one or more folders and a format:

  • JSON — full data with all metadata (best for backup)
  • CSV — spreadsheet-friendly, opens in Excel/Sheets
  • Markdown — .md file with clickable links, starred/liked badges
  • HTML Bookmarks — import into any browser
  • PDF — styled document with favicons, print directly

Global search

Press Ctrl + K (or /) to open global search. This searches link titles, URLs, and domains across all your folders.

App lock PIN

Set a 6-digit PIN in Settings → Security → App Lock PIN. When set:

  • A lock button appears in the header — tap to lock immediately
  • The app automatically locks after your chosen inactivity period (5/15/30 min)
  • Reloading within 10 minutes does not re-lock the app

Themes & fonts

In Settings → Appearance, you can:

  • Switch between Light and Dark mode
  • Choose an accent colour: Blue, Purple, Green, Orange, or Rose (applied to the logo and UI)
  • Set font size: Small, Medium, or Large

Keyboard shortcuts

  • Ctrl+K or / — Open global search
  • Esc — Close modals and overlays
  • Enter — Submit forms and dialogs
