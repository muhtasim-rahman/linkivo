# Linkivo (v1.0.0)

Linkivo is a smart, highly interactive link management web application. It allows users to extract links from various file formats, organize them into folders, and randomly open them based on a point-based algorithm.

## Features
- 📂 **Smart Extraction**: Auto-detect and extract links from text, PDF, images, JSON, HTML, ZIP, etc.
- 🗂️ **Folder Management**: Organize links into folders with PIN lock security.
- 🎲 **Random Link Opener**: Open links randomly based on user preferences (likes, favorites, blocks).
- 📱 **Responsive Design**: Mobile-first UI with a bottom navbar, and a sidebar for PC.
- 🌙 **Dark/Light Mode**: Beautiful UI with theme support.
- 🔒 **Secure**: Firebase Authentication and Realtime Database for PIN locks.
- 📶 **Offline Support**: PWA support with service workers.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Firebase (Auth, Firestore, Realtime DB, Storage)
- Lucide Icons
